# 📊 Руководство по настройке Excel таблицы для учёта пулов Uniswap

## 🎯 Обзор

Данное руководство поможет вам создать полноценную Excel таблицу для учёта пулов ликвидности Uniswap с интегрированными VBA макросами.

## 📋 Требования

- Microsoft Excel 2016 или новее (рекомендуется Excel 2019/365)
- Поддержка макросов (формат файла .xlsm)
- Базовые знания Excel
- Файл `UniswapPoolMacros.bas` из этого репозитория

## 🏗️ Пошаговая инструкция

### Шаг 1: Создание нового файла

1. Откройте Microsoft Excel
2. Создайте новую книгу
3. Сразу сохраните как **"Uniswap_Pool_Tracking.xlsm"**
   - File → Save As
   - Выберите формат: **Excel Macro-Enabled Workbook (*.xlsm)**

### Шаг 2: Создание структуры листов

Создайте следующие листы (в указанном порядке):

1. **Главная** - панель управления и сводка
2. **Пулы** - учёт всех позиций
3. **Заработанные комиссии** - учёт собранных комиссий
4. **Сводка по доходности** - аналитика и расчёты
5. **История операций** - лог всех действий
6. **Справочники** - дополнительные данные

**Как создать листы:**
- Нажмите "+" в нижней панели листов
- Переименуйте лист (двойной клик на имени)
- Или: правой кнопкой → Insert → Worksheet

### Шаг 3: Настройка листа "Главная"

Этот лист будет содержать панель управления и кнопки макросов.

#### 3.1 Создание заголовка

В ячейке **A1** введите:
```
Система учёта пулов ликвидности Uniswap
```

Форматирование:
- Шрифт: Arial, 18pt, жирный
- Цвет: RGB(0, 112, 192) - синий
- Выравнивание: по центру
- Объединить ячейки A1:F1

#### 3.2 Добавление информационной панели

Создайте таблицу сводной информации (начиная с ячейки A3):

| Ячейка | Название | Формула |
|--------|----------|---------|
| A3 | **Метрика** | |
| B3 | **Значение** | |
| A4 | Всего активных позиций | |
| B4 | | `=COUNTIF(Пулы!F:F,"Активна")` |
| A5 | Общая сумма в пулах | |
| B5 | | `=SUMIF(Пулы!F:F,"Активна",Пулы!E:E)` |
| A6 | Всего заработано (USD) | |
| B6 | | `=SUM('Заработанные комиссии'!G:G)` |
| A7 | Средняя доходность | |
| B7 | | `=IFERROR(B6/B5*100,0)&"%"` |

Форматирование:
- Ячейки A3:B3: жирный шрифт, заливка RGB(217, 217, 217)
- Столбец A: выравнивание по левому краю
- Столбец B: выравнивание по правому краю, формат числа с разделителями

#### 3.3 Добавление кнопок макросов

**Включение вкладки "Разработчик":**
1. File → Options → Customize Ribbon
2. В правой части установите галочку **Developer**
3. Нажмите OK

**Создание кнопок:**

1. Перейдите на вкладку **Developer** (Разработчик)
2. Нажмите **Insert** → **Button (Form Control)**
3. Нарисуйте кнопку в области A10:B11
4. В диалоговом окне выберите макрос **AddNewPosition**
5. Нажмите OK
6. Щёлкните правой кнопкой на кнопке → **Edit Text**
7. Введите текст: **"Добавить позицию"**

Повторите для остальных макросов:

| Позиция | Размер | Макрос | Текст кнопки |
|---------|--------|--------|--------------|
| A10:B11 | 2 строки | AddNewPosition | Добавить позицию |
| C10:D11 | 2 строки | CollectFees | Собрать комиссии |
| E10:F11 | 2 строки | ClosePosition | Закрыть позицию |
| A12:B13 | 2 строки | UpdateAllData | Обновить данные |

**Форматирование кнопок:**
- Шрифт: Arial, 10pt, жирный
- Цвет фона: белый
- Границы: чёрные

### Шаг 4: Настройка листа "Пулы"

Это основной рабочий лист для учёта позиций.

#### 4.1 Создание заголовков

В первой строке создайте следующие заголовки:

| A | B | C | D | E | F | G | H | I | J | K | L | M | N |
|---|---|---|---|---|---|---|---|---|---|---|---|---|---|
| ID | Дата открытия | Пара токенов | Диапазон цен | Сумма позиции | Статус | Token0 | Token0 Amount | Token1 | Token1 Amount | Тип пула | Fee Tier | NFT ID | Примечания |

**Форматирование заголовков:**
- Шрифт: Arial, 10pt, жирный
- Заливка: RGB(68, 114, 196) - синий
- Цвет текста: белый
- Выравнивание: по центру
- Границы: все границы, белый цвет
- Высота строки: 30
- Перенос текста: включён

#### 4.2 Настройка ширины столбцов

| Столбец | Ширина |
|---------|--------|
| A (ID) | 6 |
| B (Дата открытия) | 12 |
| C (Пара токенов) | 14 |
| D (Диапазон цен) | 14 |
| E (Сумма позиции) | 14 |
| F (Статус) | 10 |
| G (Token0) | 8 |
| H (Token0 Amount) | 14 |
| I (Token1) | 8 |
| J (Token1 Amount) | 14 |
| K (Тип пула) | 8 |
| L (Fee Tier) | 8 |
| M (NFT ID) | 10 |
| N (Примечания) | 20 |

#### 4.3 Форматирование области данных

Выделите диапазон **A1:N100** и примените:
- Границы: все границы, тонкие, серые
- Шрифт: Arial, 10pt
- Выравнивание: по центру по вертикали

#### 4.4 Условное форматирование для статуса

1. Выделите столбец **F (Статус)** (F2:F100)
2. Home → Conditional Formatting → New Rule
3. Выберите "Format only cells that contain"
4. В условии: Cell Value = "Активна"
5. Format: Заливка зелёная RGB(198, 239, 206)
6. Нажмите OK

Повторите для "Закрыта":
- Условие: Cell Value = "Закрыта"
- Format: Заливка серая RGB(220, 220, 220), текст серый RGB(100, 100, 100)

#### 4.5 Закрепление области

Закрепите первую строку:
- View → Freeze Panes → Freeze Top Row

#### 4.6 Автофильтр

Включите автофильтр для первой строки:
- Выделите A1:N1
- Data → Filter

### Шаг 5: Настройка листа "Заработанные комиссии"

#### 5.1 Создание заголовков

| A | B | C | D | E | F | G | H | I |
|---|---|---|---|---|---|---|---|---|
| ID | Дата сбора | ID пула | Сумма комиссии | Токен | Цена токена USD | Сумма USD | Способ сбора | Примечания |

**Форматирование:** аналогично листу "Пулы"

#### 5.2 Настройка ширины столбцов

| Столбец | Ширина |
|---------|--------|
| A (ID) | 6 |
| B (Дата сбора) | 12 |
| C (ID пула) | 8 |
| D (Сумма комиссии) | 16 |
| E (Токен) | 8 |
| F (Цена токена USD) | 14 |
| G (Сумма USD) | 14 |
| H (Способ сбора) | 14 |
| I (Примечания) | 20 |

#### 5.3 Формула для столбца G (Сумма USD)

В ячейке **G2** введите формулу:
```excel
=D2*F2
```

Затем скопируйте формулу вниз (протяните за правый нижний угол ячейки до G100)

#### 5.4 Форматирование чисел

- Столбец D, F, G: формат **Currency** (валюта) с 2 знаками после запятой
  - Формат: `#,##0.00`

#### 5.5 Закрепление и фильтр

Аналогично листу "Пулы":
- View → Freeze Panes → Freeze Top Row
- Data → Filter (на A1:I1)

### Шаг 6: Настройка листа "Сводка по доходности"

#### 6.1 Создание раздела "Общая статистика"

| Ячейка | Содержание | Формула |
|--------|------------|---------|
| A1 | **Общая статистика** | |
| A3 | Всего позиций: | |
| B3 | | `=COUNTA(Пулы!A:A)-1` |
| A4 | Активных позиций: | |
| B4 | | `=COUNTIF(Пулы!F:F,"Активна")` |
| A5 | Закрытых позиций: | |
| B5 | | `=COUNTIF(Пулы!F:F,"Закрыта")` |
| A6 | Общая сумма в активных пулах: | |
| B6 | | `=SUMIF(Пулы!F:F,"Активна",Пулы!E:E)` |
| A7 | Всего заработано комиссий (USD): | |
| B7 | | `=SUM('Заработанные комиссии'!G:G)` |
| A8 | Процент доходности: | |
| B8 | | `=IFERROR(B7/B6*100,0)&"%"` |

#### 6.2 Создание раздела "Доходность по пулам"

Начиная с ячейки A10:

| A | B | C | D | E |
|---|---|---|---|---|
| ID пула | Пара токенов | Сумма позиции | Заработано USD | Доходность % |

Формулы для первой строки данных (A11):
- **A11:** `=Пулы!A2` (ID пула)
- **B11:** `=Пулы!C2` (Пара токенов)
- **C11:** `=Пулы!E2` (Сумма позиции)
- **D11:** `=SUMIF('Заработанные комиссии'!C:C,A11,'Заработанные комиссии'!G:G)` (Заработано)
- **E11:** `=IFERROR(D11/C11*100,0)` (Доходность %)

Скопируйте формулы вниз на 20-30 строк.

#### 6.3 Создание графика доходности

1. Выделите диапазон B10:E20 (или сколько у вас позиций)
2. Insert → Charts → Column Chart
3. Выберите "Clustered Column"
4. Настройте заголовок: "Доходность по пулам"
5. Разместите график справа от таблицы (начиная с F10)

### Шаг 9: Импорт VBA макросов

1. Нажмите **Alt + F11** для открытия редактора VBA
2. В меню выберите **File → Import File**
3. Найдите и выберите файл **UniswapPoolMacros.bas**
4. Нажмите **Open**
5. Модуль появится в дереве проекта слева под **Modules**

### Шаг 10: Настройка безопасности макросов

1. Закройте редактор VBA (Alt + Q)
2. Перейдите в **File → Options**
3. Выберите **Trust Center → Trust Center Settings**
4. Перейдите в **Macro Settings**
5. Выберите **"Disable all macros with notification"**
6. Нажмите **OK**

### Шаг 11: Проверка работы макросов

1. Сохраните файл (**Ctrl + S**)
2. Перейдите на лист **"Главная"**
3. Нажмите кнопку **"Добавить позицию"**
4. Если появится предупреждение о безопасности, нажмите **Enable Content**
5. Подтвердите добавление позиции
6. Проверьте, что новая строка добавлена в лист "Пулы"

### Шаг 12: Защита структуры (опционально)

Для предотвращения случайного удаления формул:

1. Выделите ячейки с данными (не формулами) на каждом листе
2. Нажмите **Ctrl + 1** → вкладка **Protection**
3. Снимите галочку **Locked**
4. Перейдите в **Review → Protect Sheet**
5. Оставьте галочки только на:
   - Select locked cells
   - Select unlocked cells
   - Format cells
6. Установите пароль (необязательно)
7. Нажмите **OK**

## 🎨 Дополнительные улучшения

### Цветовая схема

Используйте единую цветовую схему для всех листов:

- **Заголовки:** RGB(68, 114, 196) - синий
- **Активный статус:** RGB(198, 239, 206) - светло-зелёный
- **Закрытый статус:** RGB(220, 220, 220) - светло-серый
- **Важные ячейки:** RGB(255, 230, 153) - светло-жёлтый
- **Формулы:** RGB(217, 225, 242) - светло-синий

### Именованные диапазоны

Создайте именованные диапазоны для упрощения формул:

1. **ActivePools** = `Пулы!$A$2:$N$100`
2. **FeeRecords** = `'Заработанные комиссии'!$A$2:$I$100`

**Как создать:**
- Выделите диапазон
- В поле Name Box (слева от строки формул) введите имя
- Нажмите Enter

### Проверка данных (Data Validation)

Добавьте выпадающие списки для некоторых полей:

**Столбец F (Статус) в листе "Пулы":**
1. Выделите F2:F100
2. Data → Data Validation
3. Allow: List
4. Source: `Активна,Закрыта,Приостановлена`
5. Нажмите OK

**Столбец K (Тип пула):**
1. Выделите K2:K100
2. Data → Data Validation
3. Allow: List
4. Source: `V2,V3,Другой`
5. Нажмите OK

**Столбец L (Fee Tier):**
1. Выделите L2:L100
2. Data → Data Validation
3. Allow: List
4. Source: `0.05%,0.30%,1.00%`
5. Нажмите OK

## 🆘 Устранение неполадок

### Макросы не работают

**Проблема:** При нажатии кнопки ничего не происходит

**Решение:**
1. Проверьте, что файл сохранён в формате .xlsm
2. Убедитесь, что макросы включены (Enable Content)
3. Проверьте настройки безопасности макросов
4. Переимпортируйте модуль VBA

### Формулы показывают ошибку #REF!

**Проблема:** Формулы не работают

**Решение:**
1. Проверьте, что все листы названы правильно
2. Убедитесь, что нет лишних пробелов в названиях листов
3. Пересоздайте формулы вручную

### Кнопки исчезли

**Проблема:** Кнопки макросов пропали с листа

**Решение:**
1. Проверьте, что не включён режим редактирования
2. Пересоздайте кнопки через вкладку Developer → Insert → Button
