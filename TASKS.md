### Этап 2: Создание формул и расчётов (Приоритет: Высокий)

#### Задача 2.1: Формулы для листа "Пулы"
- **Описание**: Добавить формулы для расчёта текущей стоимости
- **Подзадачи**:
  - [ ] Создать формулу расчёта текущей стоимости в USD
  - [ ] Настроить связь с листом "Настройки и константы" для курсов токенов
  - [ ] Протестировать формулы на примерах
- **Формула**: `=(Сумма Token A * Курс Token A) + (Сумма Token B * Курс Token B)`
- **Ожидаемый результат**: Автоматический расчёт стоимости позиций
- **Время**: 15-20 минут

#### Задача 2.2: Формулы для листа "Заработанные комиссии"
- **Описание**: Добавить формулы для расчёта комиссий в USD
- **Подзадачи**:
  - [ ] Создать формулу расчёта комиссии в USD
  - [ ] Создать формулу накопительной суммы
  - [ ] Протестировать на примерах
- **Формулы**:
  - Комиссия USD: `=(Комиссия Token A * Курс) + (Комиссия Token B * Курс)`
  - Всего собрано: `=СУММ($F$2:F2)`
- **Ожидаемый результат**: Автоматический расчёт заработанных комиссий
- **Время**: 15-20 минут

#### Задача 2.3: Формулы для листа "Транзакции"
- **Описание**: Добавить формулы для расчёта сумм операций
- **Подзадачи**:
  - [ ] Создать формулу расчёта суммы в USD
  - [ ] Протестировать на примерах транзакций
- **Формула**: `=Количество * Цена токена USD`
- **Ожидаемый результат**: Автоматический расчёт сумм транзакций
- **Время**: 10-15 минут

#### Задача 2.4: Формулы для листа "Анализ доходности" (Критично!)
- **Описание**: Создать комплексные формулы для анализа
- **Подзадачи**:
  - [ ] Формула расчёта текущей стоимости активов (связь с "Пулы")
  - [ ] Формула расчёта заработанных комиссий (связь с "Заработанные комиссии")
  - [ ] Формула расчёта непостоянных потерь (Impermanent Loss)
  - [ ] Формула расчёта Gas Fees (сумма из "Транзакции")
  - [ ] Формула расчёта чистой прибыли/убытка
  - [ ] Формула расчёта ROI (%)
  - [ ] Формула расчёта APR (%)
  - [ ] Формула расчёта количества дней в позиции
  - [ ] Протестировать все формулы на реальных примерах
- **Формулы**: См. раздел "Шаг 3: Создание формул" в UNISWAP_POOL_TRACKING.md
- **Ожидаемый результат**: Полнофункциональный лист анализа с автоматическими расчётами
- **Время**: 45-60 минут
