## 🔧 Установка и настройка

### Шаг 1: Подготовка файла Excel

1. Создайте новый файл Excel с поддержкой макросов (`.xlsm`)
2. Создайте необходимые листы:
   - **Пулы** - основной лист для учёта позиций
   - **Заработанные комиссии** - лист для учёта собранных комиссий
   - **История операций** (опционально) - для логирования действий

### Шаг 2: Настройка структуры листа "Пулы"

Создайте заголовки в первой строке листа "Пулы":

| A | B | C | D | E | F |
|---|---|---|---|---|---|
| ID | Дата открытия | Пара токенов | Диапазон цен | Сумма позиции | Статус |

**Пример заполнения:**
| ID | Дата открытия | Пара токенов | Диапазон цен | Сумма позиции | Статус |
|----|---------------|--------------|--------------|---------------|---------|
| 1  | 15.01.2026    | ETH/USDC     | 3000-3500    | 10000         | Активна |
| 2  | 18.01.2026    | WBTC/ETH     | 0.05-0.06    | 5000          | Активна |

### Шаг 3: Настройка структуры листа "Заработанные комиссии"

Создайте заголовки в первой строке листа "Заработанные комиссии":

| A | B | C | D | E |
|---|---|---|---|---|
| ID | Дата сбора | ID пула | Сумма комиссии | Токен |

**Пример заполнения:**
| ID | Дата сбора | ID пула | Сумма комиссии | Токен |
|----|------------|---------|----------------|-------|
| 1  | 16.01.2026 | 1       | 15.5           | ETH   |
| 2  | 17.01.2026 | 1       | 250.75         | USDC  |

### Шаг 4: Импорт модуля VBA

1. Откройте редактор VBA (`Alt + F11`)
2. Выберите **File → Import File** (или **Файл → Импорт файла**)
3. Выберите файл `UniswapPoolMacros.bas`
4. Модуль появится в списке модулей вашего проекта

### Шаг 5: Настройка безопасности макросов

1. Перейдите в **File → Options → Trust Center → Trust Center Settings** (Файл → Параметры → Центр управления безопасностью → Параметры центра управления безопасностью)
2. Выберите **Macro Settings** (Параметры макросов)
3. Установите **"Disable all macros with notification"** (Отключить все макросы с уведомлением)
4. Нажмите **OK**

### Шаг 6: Добавление кнопок для запуска макросов

#### Вариант 1: Через панель "Разработчик"

1. Включите вкладку "Разработчик":
   - **File → Options → Customize Ribbon** (Файл → Параметры → Настроить ленту)
   - Установите галочку напротив **Developer** (Разработчик)

2. На вкладке "Разработчик" нажмите **Insert → Button** (Вставка → Кнопка)

3. Нарисуйте кнопку на листе и в появившемся диалоге выберите макрос

4. Создайте кнопки для каждого макроса:
   - **Добавить позицию** → `AddNewPosition`
   - **Собрать комиссии** → `CollectFees`
   - **Закрыть позицию** → `ClosePosition`
   - **Обновить данные** → `UpdateAllData`

#### Вариант 2: Через Quick Access Toolbar (Панель быстрого доступа)

1. Щёлкните правой кнопкой по панели быстрого доступа
2. Выберите **Customize Quick Access Toolbar** (Настроить панель быстрого доступа)
3. В выпадающем списке выберите **Macros** (Макросы)
4. Добавьте нужные макросы на панель

## 📝 Описание макросов

### 1. AddNewPosition (Добавить позицию)

**Назначение:** Автоматически добавляет новую строку в лист "Пулы" с заполнением ID и даты.

**Что делает макрос:**
- Генерирует уникальный ID (автоинкремент на основе последнего ID)
- Автоматически заполняет текущую дату
- Устанавливает статус "Активна"
- Применяет форматирование для новой строки
- Запрашивает подтверждение перед добавлением

**Заполняемые поля:**
- ID позиции (автоматически)
- Дата открытия (текущая дата)
- Статус (по умолчанию "Активна")

**Поля для ручного заполнения:**
- Пара токенов
- Диапазон цен
- Сумма позиции

### 2. CollectFees (Собрать комиссии)

**Назначение:** Добавляет запись о сборе комиссий в лист "Заработанные комиссии".

**Что делает макрос:**
- Запрашивает ID пула
- Проверяет существование пула
- Проверяет статус пула (предупреждает, если пул неактивен)
- Запрашивает сумму комиссии
- Запрашивает токен комиссии
- Генерирует уникальный ID записи
- Заполняет текущую дату

**Проверки:**
- Существование пула с указанным ID
- Валидация суммы (должна быть больше 0)
- Предупреждение при сборе комиссий с неактивного пула

### 3. ClosePosition (Закрыть позицию)

**Назначение:** Обновляет статус позиции на "Закрыта".

**Что делает макрос:**
- Запрашивает ID позиции для закрытия
- Проверяет существование позиции
- Проверяет, не закрыта ли уже позиция
- Обновляет статус на "Закрыта"
- Применяет визуальное форматирование (серый цвет)
- Запрашивает подтверждение

**Визуальные изменения:**
- Фон строки становится серым (RGB: 220, 220, 220)
- Текст становится серым (RGB: 100, 100, 100)

### 4. UpdateAllData (Обновить все данные)

**Назначение:** Принудительный пересчёт всех формул и обновление связей.

**Что делает макрос:**
- Обновляет все внешние связи
- Выполняет полный пересчёт всех формул (`CalculateFull`)
- Обновляет сводные таблицы (если есть)
- Показывает время выполнения операции
- Запрашивает подтверждение

**Когда использовать:**
- После массового импорта данных
- При подозрении на устаревшие вычисления
- После изменения формул
- При работе с большими таблицами, где автоматический пересчёт отключён

## 📖 Подробное руководство

### Сценарий 1: Добавление новой позиции в пул

1. Нажмите кнопку **"Добавить позицию"** или запустите макрос `AddNewPosition`
2. Подтвердите добавление новой позиции (нажмите **Да**)
3. Макрос автоматически создаст новую строку с заполненными:
   - ID (например, 3)
   - Датой открытия (текущая дата)
   - Статусом ("Активна")
4. Вручную заполните оставшиеся поля:
   - **Пара токенов** (например, "ETH/USDC")
   - **Диапазон цен** (например, "3000-3500")
   - **Сумма позиции** (например, 10000)

### Сценарий 2: Сбор комиссий

1. Нажмите кнопку **"Собрать комиссии"** или запустите макрос `CollectFees`
2. Введите **ID пула**, с которого собираются комиссии (например, `1`)
3. Введите **сумму собранной комиссии** (например, `15.5`)
4. Введите **токен комиссии** (например, `ETH`)
5. Подтвердите добавление записи (нажмите **Да**)
6. Макрос добавит запись в лист "Заработанные комиссии"

**Примечание:** Если пул неактивен, макрос покажет предупреждение, но позволит продолжить.

### Сценарий 3: Закрытие позиции

1. Нажмите кнопку **"Закрыть позицию"** или запустите макрос `ClosePosition`
2. Введите **ID позиции**, которую хотите закрыть (например, `2`)
3. Подтвердите закрытие позиции (нажмите **Да**)
4. Макрос обновит статус на "Закрыта" и применит серое форматирование

**Примечание:** Если позиция уже закрыта, макрос выведет информационное сообщение.

### Сценарий 4: Обновление всех данных

1. Нажмите кнопку **"Обновить данные"** или запустите макрос `UpdateAllData`
2. Подтвердите обновление (нажмите **Да**)
3. Макрос выполнит:
   - Обновление всех связей
   - Полный пересчёт формул
   - Обновление сводных таблиц
4. По завершении появится сообщение с временем выполнения

## 🔐 Безопасность и ограничения

### Безопасность

✅ **Что БЕЗОПАСНО:**
- Все макросы запрашивают подтверждение перед выполнением критических операций
- Макросы работают только с данными (не изменяют защищённые формулы)
- Предусмотрена проверка корректности вводимых данных
- Обработка ошибок с понятными сообщениями

⚠️ **Ограничения:**
- Макросы не поддерживают отмену (Undo) для некоторых операций
- Рекомендуется создавать резервные копии перед массовыми операциями
- При закрытии Excel без сохранения все изменения будут потеряны

### Защита от ошибок

Все макросы включают:
- **Проверку существования листов** перед выполнением операций
- **Валидацию данных** (например, сумма должна быть больше 0)
- **Обработку ошибок** с информативными сообщениями
- **Подтверждение действий** перед изменением данных

### Рекомендации по безопасности

1. **Регулярно сохраняйте файл** (`Ctrl + S`)
2. **Создавайте резервные копии** перед массовыми операциями
3. **Не изменяйте код макросов** без понимания VBA
4. **Проверяйте данные** после выполнения макросов
5. **Используйте версионный контроль** для важных файлов

## 🔧 Решение проблем

### Проблема: "Лист не найден"

**Причина:** Отсутствует требуемый лист (например, "Пулы" или "Заработанные комиссии")

**Решение:**
1. Создайте лист с точным названием (учитывайте регистр)
2. Убедитесь, что лист не переименован
3. Проверьте наличие заголовков в первой строке

### Проблема: "Макрос не запускается"

**Причина:** Макросы отключены в настройках безопасности

**Решение:**
1. Перейдите в **File → Options → Trust Center → Trust Center Settings**
2. Выберите **Macro Settings**
3. Включите **"Disable all macros with notification"**
4. Перезапустите Excel и разрешите выполнение макросов

### Проблема: "Ошибка при выполнении макроса"

**Причина:** Некорректная структура данных или повреждённый файл

**Решение:**
1. Проверьте, что заголовки находятся в первой строке
2. Убедитесь, что столбцы не перемещены
3. Проверьте, что ID позиций - числовые значения
4. Попробуйте выполнить `UpdateAllData` для обновления всех данных

### Проблема: "Дублирование ID"

**Причина:** Ручное редактирование ID или ошибка при копировании

**Решение:**
1. Вручную исправьте дублирующиеся ID
2. Убедитесь, что ID идут по порядку без пропусков
3. Макрос автоматически сгенерирует следующий корректный ID

### Проблема: "Медленная работа макроса UpdateAllData"

**Причина:** Большое количество формул или сложные вычисления

**Решение:**
1. Это нормальное поведение для больших таблиц
2. Подождите завершения операции (не прерывайте)
3. Оптимизируйте формулы (замените волатильные функции)
4. Используйте `UpdateAllData` только когда это необходимо
