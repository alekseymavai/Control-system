# Алгоритм создания таблицы для учёта пулов ликвидности Uniswap

## 📋 Обзор

Данный документ описывает алгоритм создания таблицы Excel для учёта и мониторинга работы пулов ликвидности на децентрализованной бирже Uniswap.

## 🎯 Цель

Создать систему учёта, которая позволит:
- Отслеживать все активные позиции в пулах ликвидности
- Вычислять доходность и убытки
- Анализировать эффективность инвестиций
- Планировать стратегию управления ликвидностью

## 🔧 Алгоритм создания таблицы

### Шаг 1: Подготовка структуры
1. Создайте новый файл Excel
2. Создайте 5 листов с названиями: "Пулы", "Заработанные комиссии", "Транзакции", "Анализ доходности", "Настройки и константы"
3. Добавьте заголовки столбцов согласно структуре выше

### Шаг 2: Настройка форматирования
1. Примените форматирование к заголовкам (жирный шрифт, заливка цветом)
2. Установите форматы ячеек:
   - Даты: формат даты
   - Числовые значения токенов: числовой формат с достаточным количеством знаков после запятой
   - USD значения: денежный формат
   - Проценты: процентный формат
   - Адреса: текстовый формат
3. Включите автофильтр для всех таблиц

### Шаг 3: Создание формул

#### На листе "Пулы"
```excel
Текущая стоимость USD =
  (Сумма Token A * Курс Token A) + (Сумма Token B * Курс Token B)
```

#### На листе "Заработанные комиссии"
```excel
Комиссия USD =
  (Комиссия Token A * Курс Token A) + (Комиссия Token B * Курс Token B)

Всего собрано USD =
  СУММ($F$2:F2)  // накопительная сумма
```

#### На листе "Транзакции"
```excel
Сумма USD = Количество * Цена токена USD
```

#### На листе "Анализ доходности"

**Формула непостоянных потерь (Impermanent Loss):**
```excel
// Для расчёта IL нужны начальная и текущая цены
Текущая стоимость HODL = Начальные инвестиции USD
Текущая стоимость в пуле = Текущая стоимость активов USD + Заработанные комиссии USD
Непостоянные потери USD = Текущая стоимость в пуле - Текущая стоимость HODL
```

**Чистая прибыль/убыток:**
```excel
Чистая прибыль =
  (Текущая стоимость активов USD + Заработанные комиссии USD) -
  Начальные инвестиции USD -
  СУММЕСЛИ(Транзакции!D:D, ID позиции, Транзакции!I:I)  // Gas fees
```

**ROI (Return on Investment):**
```excel
ROI (%) = (Чистая прибыль USD / Начальные инвестиции USD) * 100
```

**APR (Annual Percentage Rate):**
```excel
APR (%) = (ROI / Дней в позиции) * 365
```

### Шаг 4: Условное форматирование
1. **На листе "Пулы"**, столбец "Статус":
   - Зелёный: "В диапазоне"
   - Красный: "Вне диапазона"
   - Серый: "Закрыта"

2. **На листе "Анализ доходности"**:
   - Столбец "Чистая прибыль/убыток USD":
     - Зелёный фон при значении > 0
     - Красный фон при значении < 0
   - Столбец "ROI (%)":
     - Градиент: красный → жёлтый → зелёный

3. **Предупреждения**:
   - Выделить ячейку, если непостоянные потери > порога из настроек

### Шаг 5: Создание сводной панели (опционально)
На первом листе создайте Dashboard с ключевыми метриками:
- Общая стоимость всех позиций
- Всего заработано комиссий
- Средний ROI
- Количество активных позиций
- График доходности по времени

### Шаг 6: Автоматизация (продвинутый уровень)
1. **Использование Power Query** для импорта данных из Uniswap API или Etherscan
2. **Макросы VBA** для автоматического обновления курсов токенов
3. **Интеграция с Google Sheets** для работы совместно с командой

## 📝 Формулы для расчёта Impermanent Loss (для V2)

### Точная формула IL для Uniswap V2:
```
IL = 2 * sqrt(PriceRatio) / (1 + PriceRatio) - 1

где PriceRatio = Текущая цена / Начальная цена
```

### Пример в Excel:
```excel
// В ячейке "Изменение цены"
=L2/K2  // Текущая цена / Начальная цена

// В ячейке "Impermanent Loss %"
=(2*КОРЕНЬ(M2)/(1+M2)-1)*100

// В ячейке "Непостоянные потери USD"
=N2/100*Начальные_инвестиции
```

## 🔄 Процесс работы с таблицей

### При добавлении ликвидности:
1. Добавьте новую строку в лист "Пулы"
2. Заполните все поля о позиции
3. Добавьте транзакцию в лист "Транзакции" (тип: "Добавление")
4. Проверьте, что формулы на листе "Анализ доходности" подхватили новую позицию

### При сборе комиссий:
1. Добавьте запись в лист "Заработанные комиссии"
2. Добавьте транзакцию в лист "Транзакции" (тип: "Сбор комиссий")
3. Обновите данные в листе "Анализ доходности"

### При удалении ликвидности:
1. Обновите статус позиции на "Закрыта" в листе "Пулы"
2. Добавьте транзакцию в лист "Транзакции" (тип: "Удаление")
3. Зафиксируйте финальную доходность в листе "Анализ доходности"

### Регулярное обновление (ежедневно/еженедельно):
1. Обновите курсы токенов в листе "Настройки и константы"
2. Обновите "Текущая цена" в листе "Пулы"
3. Проверьте статус позиций (в диапазоне/вне диапазона для V3)
4. Оцените текущую доходность в листе "Анализ доходности"
