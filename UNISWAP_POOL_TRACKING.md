# 📊 Система учёта пулов ликвидности Uniswap

## Введение

Данная система предназначена для отслеживания и анализа позиций в пулах ликвидности на децентрализованной бирже Uniswap. Система построена на базе Microsoft Excel и предоставляет визуальный Dashboard для мониторинга ключевых метрик.

---

## 🎯 Основные возможности

### 1. Учёт позиций в пулах ликвидности
- Отслеживание всех активных и закрытых позиций
- Учёт количества токенов в каждой паре
- Автоматический расчёт текущей стоимости позиций
- Мониторинг заработанных комиссий

### 2. Анализ доходности
- Временные графики накопленных комиссий
- Отслеживание изменения стоимости портфеля
- Расчёт ROI (Return on Investment)
- Анализ эффективности различных пар

### 3. Визуальный Dashboard
- Ключевые метрики в одном месте
- Интерактивные графики и диаграммы
- Автоматическое обновление данных
- Профессиональный дизайн

## 📋 Структура данных

### Лист "Пулы"

| Поле | Описание | Тип данных | Пример |
|------|----------|------------|--------|
| ID Позиции | Уникальный идентификатор | Число | 1, 2, 3... |
| Пара | Торговая пара токенов | Текст | ETH/USDC |
| Token 0 | Первый токен в паре | Текст | ETH |
| Token 1 | Второй токен в паре | Текст | USDC |
| Количество Token 0 | Количество первого токена | Число | 2.5 |
| Количество Token 1 | Количество второго токена | Число | 5000 |
| Цена Token 0 (USD) | Текущая цена в USD | Валюта | $2000.00 |
| Цена Token 1 (USD) | Текущая цена в USD | Валюта | $1.00 |
| Общая стоимость (USD) | Автоматически рассчитывается | Валюта | $10,000.00 |
| Дата открытия | Дата создания позиции | Дата | 2025-12-15 |
| Статус | Активна/Закрыта | Текст | Активна |
| Fee Tier (%) | Уровень комиссии пула | Процент | 0.3% |
| Заработано комиссий (USD) | Накопленные комиссии | Валюта | $150.00 |
| ROI (%) | Возврат инвестиций | Процент | 1.50% |

### Лист "Анализ доходности"

| Поле | Описание | Формула |
|------|----------|---------|
| Дата | Дата записи | - |
| Накопленные комиссии (USD) | Общая сумма комиссий | - |
| Общая стоимость портфеля (USD) | Текущая стоимость всех позиций | - |
| ROI (%) | Процент доходности | `=(Комиссии/Портфель)*100` |

### Лист "Dashboard"

#### Ключевые метрики

1. **Общая стоимость всех позиций**
   - Формула: `=SUM(Пулы!I:I)`
   - Описание: Суммарная стоимость всех позиций в USD
   - Использует: Столбец "Общая стоимость" из листа "Пулы"

2. **Всего заработано комиссий**
   - Формула: `=SUM(Пулы!M:M)`
   - Описание: Общая сумма заработанных комиссий
   - Использует: Столбец "Заработано комиссий" из листа "Пулы"

3. **Средний ROI**
   - Формула: `=AVERAGE(Пулы!N:N)`
   - Описание: Средняя доходность по всем позициям
   - Использует: Столбец "ROI (%)" из листа "Пулы"

4. **Количество активных позиций**
   - Формула: `=COUNTIF(Пулы!K:K,"Активна")`
   - Описание: Подсчёт позиций со статусом "Активна"
   - Использует: Столбец "Статус" из листа "Пулы"

#### Визуализации

1. **График доходности по времени**
   - Тип: Линейный график
   - Данные: Накопленные комиссии из листа "Анализ доходности"
   - Назначение: Отслеживание роста доходов во времени

2. **Круговая диаграмма распределения**
   - Тип: Круговая диаграмма
   - Данные: Стоимость позиций по парам из листа "Пулы"
   - Назначение: Визуализация распределения капитала

## 🔢 Формулы и расчёты

### Расчёт общей стоимости позиции

```excel
=E2*G2+F2*H2
```

Где:
- E2 = Количество Token 0
- G2 = Цена Token 0 (USD)
- F2 = Количество Token 1
- H2 = Цена Token 1 (USD)

**Пример:**
- 2.5 ETH × $2000 + 5000 USDC × $1 = $5000 + $5000 = $10,000

### Расчёт ROI

```excel
=(M2/I2)*100
```

Где:
- M2 = Заработано комиссий (USD)
- I2 = Общая стоимость (USD)

**Пример:**
- ($150 / $10,000) × 100 = 1.5%

## 📈 Ключевые метрики пулов Uniswap

### Fee Tiers (Уровни комиссий)

Uniswap v3 поддерживает несколько уровней комиссий:

| Fee Tier | Описание | Типичное использование |
|----------|----------|------------------------|
| 0.01% | Очень низкая | Стабильные пары (USDC/DAI) |
| 0.05% | Низкая | Коррелированные активы |
| 0.30% | Средняя | Большинство пар (ETH/USDC) |
| 1.00% | Высокая | Экзотические/волатильные пары |

### Impermanent Loss (IL)

**Важно:** Данная система не учитывает непостоянные потери (impermanent loss) автоматически. Для полного анализа рекомендуется:

1. Отслеживать начальное соотношение токенов
2. Сравнивать с текущим соотношением
3. Рассчитывать потенциальные потери от переоценки

### ROI vs APR/APY

- **ROI** - Return on Investment (абсолютная доходность)
- **APR** - Annual Percentage Rate (годовая процентная ставка)
- **APY** - Annual Percentage Yield (годовая доходность с учётом реинвестирования)

Система рассчитывает ROI. Для расчёта APR/APY необходимо учитывать временной период.

## 🛠️ Использование системы

### Начальная настройка

1. **Откройте файл** `Uniswap_Pool_Dashboard.xlsx`

2. **Настройте лист "Пулы":**
   - Удалите примеры данных (строки 2-6)
   - Добавьте свои позиции

3. **Обновите цены токенов:**
   - Вручную введите текущие цены в USD
   - Или настройте автоматический импорт (см. Задачу 4)

### Добавление новой позиции

1. Перейдите на лист "Пулы"
2. Добавьте новую строку в конец таблицы
3. Заполните данные:
   ```
   ID: [следующий номер]
   Пара: [TOKEN0/TOKEN1]
   Token 0: [символ токена]
   Token 1: [символ токена]
   Количество Token 0: [сумма]
   Количество Token 1: [сумма]
   Цена Token 0 (USD): [текущая цена]
   Цена Token 1 (USD): [текущая цена]
   Дата открытия: [дата]
   Статус: Активна
   Fee Tier (%): [0.01, 0.05, 0.3, или 1.0]
   Заработано комиссий (USD): 0
   ```
4. Формулы в столбцах "Общая стоимость" и "ROI" рассчитаются автоматически

### Обновление комиссий

Периодически обновляйте столбец "Заработано комиссий (USD)" на основе данных из Uniswap:

1. Зайдите в свой аккаунт на Uniswap
2. Проверьте накопленные комиссии для каждой позиции
3. Обновите значения в таблице
4. Dashboard автоматически пересчитает метрики

### Закрытие позиции

При выходе из пула:

1. Найдите соответствующую строку
2. Измените статус с "Активна" на "Закрыта"
3. Убедитесь, что финальные комиссии учтены
4. Счётчик активных позиций обновится автоматически

## ❓ FAQ

### Как часто нужно обновлять цены токенов?

Рекомендуется обновлять цены ежедневно или еженедельно, в зависимости от волатильности рынка. Для автоматического обновления см. Issue #4.

### Учитывается ли непостоянная потеря (IL)?

Текущая версия не рассчитывает IL автоматически. Планируется добавить в будущих версиях.

### Можно ли отслеживать позиции на других DEX?

Да, система универсальна и может использоваться для SushiSwap, PancakeSwap и других AMM. Просто адаптируйте названия и структуру данных.

### Как экспортировать данные для налоговой отчётности?

Используйте стандартные функции Excel для экспорта в CSV или PDF. Для автоматизации планируются макросы VBA (Issue #5).
